# AluguelDeChácaras.com.br

Plataforma de aluguel de chácaras para eventos em Maringá e região do Paraná.

## Páginas

| Arquivo | Descrição |
|---|---|
| `index.html` | Página inicial com busca e filtros |
| `chacara.html` | Página individual de cada chácara |
| `fornecedor.html` | Página individual de cada fornecedor |
| `painel-proprietario.html` | Painel de gestão do proprietário |
| `painel-fornecedor.html` | Painel de gestão do fornecedor |

## Como usar

Abra o arquivo `index.html` no navegador ou hospede no GitHub Pages.

## Estrutura do projeto

- Proprietários de chácaras anunciam com fotos, vídeos e calendário de disponibilidade
- Fornecedores de eventos (DJ, buffet, decoração, etc.) se cadastram por categoria
- Visitantes encontram chácaras e fornecedores gratuitamente
- Contato direto entre cliente e proprietário/fornecedor

## Desenvolvido com

HTML, CSS e JavaScript puro — sem dependências externas.
